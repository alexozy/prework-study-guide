HTML

  • The head element contains information about the webpage. Head is for the BROWSER.
  • The body element represents the visible content shown to the user. Body is for the USER.
  • Body element contains child & parent elements. Parent elements house the children elements
  • Each element has an opening/closing tag. Closing tags are indicated with a forward slash "/"
  • The HTML code follows a particular structue:
  • Element = the entire element from open to close
  • Tag + refers to what appears inside the angle brackets
  • Headings are represeted with angle brackets and h followed by the numbers 1-6
  • Paragraph or "p" inside angle brackets
  • For Bulleted or Numbered Lists Use: ul (unordered lists), ol ( ordered lists), li (list items), inside of angle brackets

CSS

  • CSS or Cascading Style Sheets is a progamming lang that defines how webpage content is presented to users. It defines: colors, aesthetics, visual layout, and organization. It defines style rules for specfic elemnts of an HTML page.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A declaration contains 2 components 1) A CSS Property 2) The Value of that Property i.e. color: blue| where color is the property and blue is the value.
  • What is DRY code? Do Not Repeat Yourself Code. This is code that is shorter and more conscise to reduce the amount of code used.
  • What are the components of the Box Model? From the outside in: Margin, Border, Padding, Content.
  • There are 3 Ways to Style CSS: The first is INLINE CSS which is used to style an element directly in the HTML file. Good for quick and permanent changes. The second is INTERNAL CSS Style Sheet The third and most common is External CSS Style Sheet --which was practiced in module 3.2.5-- that must be linked to the html document in order to be seen on the webpage.

Git

  • git status: checks what branch we are currently on
  • git log: allows you to view all previous commands
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit: saves new versions of your project to the repo

  • BACKTRACKING in Git
  • git show HEAD: show the most recently made commit
  • git checkout HEAD filename: reverts changes you made to a file name
